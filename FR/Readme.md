# Flag Register (FR)

A 5-bit loadable Flag Register with asynchronous reset. The Flag Register stores the status flags generated by the Arithmetic Logic Unit (ALU) and updates them on the positive edge of the clock when the `load` signal is asserted.

## Features

- Stores Carry, Negative, Greater Than, Equal and Zero flags
- Asynchronous active-high reset
- Synchronous flag loading
- Holds previous flag values when `load = 0`
- Intended for conditional branching and status monitoring
